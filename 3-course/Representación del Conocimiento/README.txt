
*** REPRESENTACIÓN DEL CONOCIMIENTO ***

Representación del Conocimiento es una asignatura en la que se explican sistemas basados en reglas.
En esta asignatura aprenderemos a programar mediante programación declarativa, usando para ello
Prolog, un sistema basado en reglas y mediante el cuál realizaremos ciertas "preguntas" para que 
nos las resuelva. Es un sistema booleano, el cual responde a las preguntas con "verdadero" o "falso".

En este repositorio, realizaremos distintos ejercicios mediante los cuales comprenderemos la programación
declarativa. El proceso de programar, es completamente diferente a todo lo que hemos aprendido anteriormente
en la programación iterativa. Para ello, usaremos el Principio de Inducción Matemática, mediante el cual
resolveremos con código todos los problemas que veremos.
